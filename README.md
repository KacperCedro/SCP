## How it works?
I relly want to know how... So how it works? it's preety simple: 
- first user should add some equalation (or just want to know 0 equal 0)
- then those numbers are converted from infix to postfix (RPN) using "shooting yard" algorythm
- next the postfix value is converted once again, but this time it will be the the final result
- finally user gets result (or error ;]
